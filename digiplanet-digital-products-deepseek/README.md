# Digiplanet Digital Products

A complete digital product sales solution for WordPress with Stripe and Paystack integration.

## Features

- **Complete E-commerce System**: Products, cart, checkout, orders
- **Dual Payment Gateways**: Stripe and Paystack integration
- **Elementor Integration**: 5+ custom widgets
- **Dual User Portals**: Separate portals for digital customers and software clients
- **License Management**: Automated license key generation and tracking
- **Secure Downloads**: Token-based secure file downloads
- **Reviews & Ratings**: Customer review system
- **Sales Analytics**: Track performance and revenue
- **Built-in Documentation**: Complete help system

## Requirements

- WordPress 5.8 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher
- cURL extension
- mbstring extension
- JSON extension

## Installation

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Digital Products â†’ Settings to configure
4. Set up payment gateways (Stripe and/or Paystack)
5. Create your digital products

## Quick Start

1. **Configure Settings**: Set currency, payment methods, and email settings
2. **Create Products**: Add digital products with files and pricing
3. **Set Up Pages**: Use provided shortcodes for cart, checkout, and account pages
4. **Test System**: Make a test purchase to verify everything works

## Shortcodes

- `[digiplanet_products]` - Display product grid
- `[digiplanet_cart]` - Show shopping cart
- `[digiplanet_checkout]` - Display checkout form
- `[digiplanet_account]` - Show user account portal
- `[digiplanet_product_search]` - Add product search form
