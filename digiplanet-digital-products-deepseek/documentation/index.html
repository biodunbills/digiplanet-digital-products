Getting Started

Welcome to Digiplanet Digital Products plugin documentation. This plugin allows you to sell digital products (scripts, themes, plugins, etc.) on your WordPress website with ease.

Installation

Requirements

  • WordPress 5.8 or higher
  • PHP 7.4 or higher
  • MySQL 5.6 or higher
  • Elementor Pro (optional, for widgets)

Installation Steps

  1. Go to WordPress Admin → Plugins → Add New
  2. Click "Upload Plugin" and choose the zip file
  3. Click "Install Now"
  4. Activate the plugin
  5. Go to Digital Products → Settings to configure
Note: The plugin will automatically create necessary database tables and default pages.

Configuration

General Settings

Configure your store settings from Digital Products → Settings → General:

  • Currency: Set your store currency
  • Currency Position: Choose where currency symbol appears
  • Tax Settings: Enable tax and set tax rate

Payment Gateways

Configure payment methods from their respective tabs:

Stripe Configuration

  1. Get your API keys from Stripe Dashboard
  2. Enable Stripe in settings
  3. Enter Test/Live keys
  4. Configure webhook URL

Paystack Configuration

  1. Get your API keys from Paystack Dashboard
  2. Enable Paystack in settings
  3. Enter Test/Live keys

Managing Products

Adding Products

  1. Go to Digital Products → Products → Add New
  2. Enter product details (name, description, price)
  3. Upload product files
  4. Set product categories and tags
  5. Configure download limits and license type
  6. Publish the product

Product Categories

Organize products using categories and subcategories from Digital Products → Categories.

Product Attributes

  • Price: Regular price of the product
  • Sale Price: Discounted price (optional)
  • Download File: Upload the digital product file
  • Download Limit: Number of allowed downloads (0 = unlimited)
  • License Type: Single, Multi-site, or Developer license
  • Requirements: System requirements for the product

Elementor Widgets

The plugin includes several Elementor widgets for easy product display:

Available Widgets

  • Product Grid: Display products in grid layout
  • Product Carousel: Product slider/carousel
  • Product Categories: Show product categories
  • Product Search: Add product search functionality
  • Add to Cart Button: Direct add to cart button

Using Widgets

  1. Edit page with Elementor
  2. Search for "Digiplanet" widgets
  3. Drag and drop widget to page
  4. Configure widget settings
  5. Save and preview

User Portal

User Roles

  • Digital Customer: Can register, purchase products, view orders and licenses
  • Software Client: Created by admin, access to client portal with support and documents

Portal Features

Digital Customers can:

  • View purchased products
  • Download products
  • View order history
  • Manage licenses
  • Submit reviews

Software Clients can:

  • Access project documents
  • Submit support tickets
  • View project progress
  • Access billing information

Troubleshooting

Common Issues

Payment Not Working

  • Check API keys are correct
  • Verify webhook URL is set correctly
  • Ensure currency matches payment processor

Downloads Not Working

  • Check file permissions
  • Verify download limits aren't exceeded
  • Check if license is active

Elementor Widgets Missing

  • Ensure Elementor is installed and activated
  • Clear WordPress cache
  • Re-save permalinks

Frequently Asked Questions

Can I use both Stripe and Paystack?

Yes, you can enable both payment gateways and customers can choose during checkout.

How do I add custom fields to products?

Use the "Requirements" field or contact support for custom field development.

Can I sell subscription products?

Currently, the plugin supports one-time purchases only. Subscription support is planned for future updates.

How do I customize email templates?

Copy email templates from plugin/templates/email to your theme folder and modify them.

Is the plugin translation ready?

Yes, the plugin uses WordPress translation system. You can find .pot file in languages folder.
